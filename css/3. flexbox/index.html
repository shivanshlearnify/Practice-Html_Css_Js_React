Topic flexbox

justify content work on main axis(main axis in which direction is set)
and align items work on cross axis(oppostite of direction defined)

if we have multiple line content in a container
if we use align-item center it will create space in between to get rid of space we use align-content

gap, row-gap, column-gap --> gap: for row, for column(shorthand)

Property used in flexbox for items

to change the order of any item we can use (order:any number;) smaller number order comes first larger goes in last

flex grow:1 By this property if space is available in main axis it will increase its width to cover its whole width
if we have two items for one flex-grow is 1 and for second flex-grow is 2 with 2 it will have double space cmparison with 1

we have flex-shrink when we make our screen in smaller size it will decrease it width
flex-shrink:0 means it will prevent it from shrinking
flex-shrink:1 means it will proportionally with the other flex items
